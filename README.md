# 🏡 Room Management Frontend
## 📖 Introduction
This project is frontend of **Room Management System**, built using **React + Vite** with TypeScript.

## 🚀 Technologies Used
- **React** + **Vite**
- **TypeScript**
- **Tailwind CSS** 


## 📦 How to use
1. Install dependencies:
   ```sh
   npm install
   ```
   or
   ```sh
   yarn install
   ```

2. Run the project:
    ```sh
    npm run dev
    ```
    or
    ```sh
    yarn dev
    ```
